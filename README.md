# ottc_ios
Software OTTC para iOS

Adicionar ao arquivo Config.plist as seguintes chaves:

Host - servidor que vai receber as posições dos motoristas

SACKey - chave para acesso ao serviço de atendimento ao cidadão

SACHost - servidor rodando o serviço de atendimento ao cidadão

OneSignalAppId - app id do OneSignal para envio de mensagens push
